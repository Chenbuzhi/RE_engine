<p>Email:   leobuzhi@gmail.com</p>
<p>A simple Regular Expression engine</p>
<p>OS:					
	Linux,*nix</p>			
<p>Process:				
	Re->NFA->DFA</p>		
<p>Algorithm:
	Re->NFA	
	Thompson algorithm		
	NFA->DFA			
	Subset Construction Method</p>	
<p>Example:				
	re:ab(b|c)a*			
	string:dabbd success  		
	result:abb   success		
	string:qabcaaaaa		
	result:abb   success		
	string:sabaaaaa			
	result:      failure</p>
