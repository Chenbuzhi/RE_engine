<p>Author:  leobuzhi</p>
<p>Email:   leobuzhi@gmail.com</p>
<p>A simple Regular Expression engine</p>
<p>OS:					<br/>
	Linux,*nix</p>			<br/>
<p>Process:				<br/>
	Re->NFA->DFA</p>		<br/>
<p>Algorithm:				<br/>
	Thompson algorithm		<br/>
	Re->NFA				<br/>
	NFA->DFA			<br/>
	Subset Construction Method</p>	<br/>
<p>Example:				<br/>
	re:ab(b|c)a*			<br/>
	string:dabbd success  		<br/>
	result:abb   success		<br/>
	string:qabcaaaaa		<br/>
	result:abb   success		<br/>
	string:sabaaaaa			<br/>
	result:      failure</p>	<br/>
